# Payment Gateway Simulator (C)

This project simulates a basic payment gateway system using pure C and in-memory data structures.

## Features
- User and merchant creation
- Payment processing
- Idempotent transaction handling
- Balance validation
- In-memory storage (no database)

## Tech Stack
- Language: C
- Data Structures: Arrays, Structs
- Concepts: Idempotency, Failure Handling

## How to Run
gcc payment_gateway.c
./a.out
